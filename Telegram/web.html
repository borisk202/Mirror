    Уведомление о использовании неофициальной версии клиента (далее "приложение") для мессенджера Telegram.

  • Приложение построено на базе версии WebK и работает через Telegram API.
  • Благодаря технологии PWA (progressive web app) приложение использует минимум ресурсов и кэшируется в браузере пользователя, а вся информация между пользователем и Telegram безопасно передается в зашифрованном виде.
  • Приложение не контролирует передаваемую информацию через Telegram API, поэтому пользовательские данные не обрабатываются и не хранятся на серверной стороне приложения. Подробнее о политике конфиденциальности приложения.
  • Приложение неофициальное и предоставляется на условиях «как есть». Пользователь приложения также принимает пользовательское соглашение с Telegram.
  • Вы можете продолжить использовать это неофициальное приложение или выбрать одну из официальных версий на сайте - https://telegram.org/apps.
  • Первичную регистрацию пользователь может совершить только в официальной мобильной версии Telegram.
